# Autonomous Youtube Analysis Agent

This project is an autonomous agent that can analyze and improve YouTube videos.

## Major Parts

*   **Frontend:** The user interface for interacting with the agent.
*   **Generator Agent:** The agent responsible for generating analysis and improvements.
*   **Reflector Agents:** Agents that provide feedback and reflection on the generated content.
*   **Curator Agent:** An agent that curates and selects the best content.
*   **Context Engineering Pipeline:** The pipeline for processing and preparing data for the agents.
*   **Tools:** A collection of tools used by the agents.
